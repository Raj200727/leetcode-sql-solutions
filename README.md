# SQL Practice & LeetCode Solutions

This repository contains my solutions to SQL and database-related problems.
The focus is on understanding joins, filtering, aggregation, and relational logic.

## Topics Covered
- SELECT and WHERE
- INNER JOIN and LEFT JOIN
- Self Joins
- Filtering with conditions
- Basic aggregation

## Folder Structure
- `easy/` – Beginner SQL problems
- `notes/` – Personal notes and explanations

## Goal
To build a strong foundation in SQL by solving problems and documenting patterns
that frequently appear in exams and interviews.
